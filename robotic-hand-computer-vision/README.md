# 🤖 Robotic Hand with Computer Vision
**Mechatronics Capstone Project (TCC)**

### 🎯 Overview
This project consists of a mechanical hand controlled by real-time human gestures. A webcam captures hand movements, Python processes the image using computer vision, and sends coordinates to an Arduino to move servomotors.

### 🛠️ Hardware & Tools
- **Microcontroller:** Arduino Uno.
- **Actuators:** 5x Servomotors.
- **Circuit:** Protoboard, jumper wires, and external power supply.

### 💻 Software Stack
- **Python 3.x**
- **OpenCV & MediaPipe:** For hand landmark tracking.
- **PySerial:** For PC-to-Arduino communication.
